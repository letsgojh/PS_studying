# 18808 : 스티커 붙이기 (골드 3)
## 코드 설명

1. sticker 배열 K개 만들어서 스티커 저장
2. 첫번째 스티커부터 위치시키면서 backtracking
3. 만약 왼쪽 위에 스티커 붙일 수 있으면 거기 위치 고정 (0도, 90도, 180도 270도 전부 다 검사)
4. 다음 스티커 위치도 똑같이 검사 반복
5. 모든 스티커 검사 했으면 스티커 갯수 세기

## 애로사항
- 처음에는 문제를 잘못 이해해서 스티커가 가장 많은 칸을 채울 수 있는 경우를 찾도록 만들었음
- 그래서 처음부터 문제를 다시 읽어보았더니 가장 왼쪽 위에 스티커를 붙일 수 있는 공간이 있으면 반드시 스티커를 붙이고 넘어가야한다는 조건을 알게되어 코드 변경함
