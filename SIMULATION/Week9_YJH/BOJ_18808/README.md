# 18808 : 스티커 붙히기

## 코드 설명

1. 좌표 하나하나씩 다 탐색하며 해당 좌표에 스티커를 놓을 수 있는지 canAttach로 확인한다(미리 N- sticker_row, M - sticker_col 조건을 둚으로써 나가는 것 방지) => canAttach true 값이라면 attach를 통해 놓기
2. 좌표 다 돌았는데도 둘 곳이 없다면 회전 후 다시 돌기
3. 전부 다 놓은 뒤 graph탐색하며 1로 표시되어 있는 곳 출력
