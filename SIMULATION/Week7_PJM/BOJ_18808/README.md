### 문제 설명(스티커 붙이기 - 18808)- 골드 3
 > N*M 크기의 노트북에 스티커를 붙이되 규칙대로 붙여야 함. 그렇게 붙였을 때 최종적으로 스티커가 붙여진 공간을 계산하시오
 > ***규칙은 다음과 같음***
 > 1. 스티커를 떼서 가장 위쪽에서 가장 왼쪽을 우선적으로 붙여야함. 이후 못 붙일 경우 오른쪽으로 이동
 > 2. 가장 위쪽에 못 붙일 경우 아래로 한칸이동후 가장 왼쪽부터 붙여야함.
 > 3. 그렇게 최종적으로 못붙일 경우 스티커를 0, 90, 180, 270도 순으로 돌려서 1~3번 반복
 > 4. 그래도 못붙이면 그 스티커는 기각

 ### 코드
 1. 스티커를 입력받을 때 0,90,180,270도로 회전했을 때의 스티커 모양을 저장함
 2. isStick() 함수를 통해 해당 시작위치에서 스티커를 붙일 수 있다면 붙이고 true를 반환하여 다음 스티커 인덱스로 넘김
 3. 못 붙일 경우 먼저 col을 증가시키고 그래도 안되면 row를 증가시키고 그래도 안되면 dir를 증가시켜서 모든 경우의수를 진행해봄.
 4. 그렇게 모두 계산한 이후에 붙인 위치를 파악해서 출력
