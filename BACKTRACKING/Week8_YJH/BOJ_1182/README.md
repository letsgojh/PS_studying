# 1182 : 부분수열의 합
## 코드 설명

1. 먼저 종료 조건을 생각한다(n이 S가 되면 끝이라고 생각했는데, 그럼 그냥 n==S인 n 들어오면 끝, 끝까지 탐색했는지도 조건으로 추가)
2. n에 지금까지 집합으로 포함하여 더한 숫자, tmp에 방문 횟수를 기록
3. for 배열을 돌면서 했던 이전 문제들과 다르게, 부분수열은 포함 되어있는지, 안되어있는지에 따라 구성
4. 노드의 왼쪽에는 포함 안된경우, 노드의 오른쪽에는 포함 된경우로 생각한다.
5. 종료조건시 ans에 1더하기
6. 만약 S가 0값이라면 공집합도 무조건 포함하기 때문에 ans-1로 출력하기

## 애로 사항
1. 마지막에 공집합빼려고 ans-- 하는게 보기싫어서 그냥 종료조건에다가 0일경우 ans++하지 않도록 했는데, 그러면 부분집합의 합이 0인 경우는 전부 pass하기 때문에 수정..
