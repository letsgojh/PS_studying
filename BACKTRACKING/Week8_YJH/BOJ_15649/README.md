# 15649 : N과 M(1)
## 코드 설명

1. 먼저 종료 조건을 생각한다(자리수가 M번째가 되면 종료)
2. 방문을 표시하는 visited 배열과 종료를 표시하는 result 배열을 생성한다(max가 8까지니깐 9번째 배열까지만 생성)
3. 인자인 tmp는 자릿 수 값을 담는다.
4. tmp == M+1이면 두개를 출력하고 종료한다.
5. 반복문으로 1부터 N까지 돌며 result[tmp] 에 해당값을 넣는다.
6. recursive로 들어가면 그다음에도 정상적으로 돌도록 visited[i] = false; 처리를 해준다.
