### 문제 설명(말이 되고픈 원숭이 - 1600 ) - 골드 3
> 체스 나이트처럼 움직일 수 있는 최댓값 K와 N*M 크기의 장애물 맵이 주어진다. (0,0)에서 (N-1,M-1) 까지 가는 최소값을 구하시오
### 코드
- 예전에 풀었던 벽뚫기와 유사함
1. visited 확인 배열을 만들되, 3차원으로 만들고 K크기 만큼 만들어줘야 함.
2. BFS를 통해 경로를 찾으면서도, 나이트처럼 움직일 수 있으면 hx,hy 배열을 통해 나이트로 움직이는 경우도 모두 확인
3. 그 외에는 다른 BFS 확인하는 것 처럼 dx,dy를 이용하여 경우 확인
4. 큐에는 x,y,말처럼 움직인 횟수, 여기 까지 온 횟수를 넣는다.
