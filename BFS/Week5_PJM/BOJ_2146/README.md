### 문제 설명(다리 만들기 - 2146 ) - 골드 3
> 섬끼리 다리를 이었을때 제일 작은 다리의 길이가 몇인가?
### 코드
1. 먼저 같은 섬인지 파악하는 BFS를 돌리고 MAP에 같은 넘버링을 새겨준다.
2. 넘버링기준으로 같은 섬을 모두 큐에 집어 넣고 BFS를 돌리기 시작한다
3. 만약 같은 넘버링이면 다리 길이는 그대로 가져가되, 해당하는 노드를 큐에 집어넣는다.
4. 만약 물이라면, 해당 노드를 집어넣되, 다리길이를 1 증가시킨다.
5. 만약 다른 넘버링이면 다른 섬에 도착했기 때문에 최소값 확인 후 갱신한다.
