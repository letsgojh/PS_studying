# 2146 : 다리 만들기
## 코드 설명

1. 먼저 distinguish_island로 새로운 island 배열 만들어 각 섬에 번호 붙히기(bfs로)
2. graph 전체를 순회하며, 각 섬마다 한번씩의 bfs만 돌리기 ex) 1번에 해당하는 섬들의 배열값을 queue에 모두 넣은다음 bfs를 한번만 돌리기 -> 원래는 모든 섬의 부분들에서 bfs를 돌렸었다.
+ 항상 distance를 기록하는 새로운 2차원 배열을 만들기. 
3. 4방향으로 탐색 중, 섬의 영역이 아니고 방문도 안되어 있을 경우에는 Queue에 넣기
4. 섬의 영역인데 현재 해당하는 섬이 아닌 경우 return
5. 이 값들 중 가장 작은 값을 출력한다.
