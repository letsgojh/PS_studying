# 2206 : 벽 부수고 이동하기
## 코드 설명
1. 거리를 측정하는 배열인 distance배열을 3차원으로 만들어 한 차원에는 벽을 뚫었을경우, 한 차원에는 벽을 뚫지 않았을 경우를 입력
2. 만약 graph값에서 0(이동할 수 있다)인 경우 distance[0]을 활용하여 이동
3. graph값에서 1(벽)인 경우 distance[1]을 활용하여 이동
4. bfs를 돌리며 다시 queue에 좌표 값을 넣을때에 graph 값이 0인 경우는 3차원 부분의 값을 이전에 꺼낸 값으로 넣으면되지만
5. graph 값이 1인경우(벽)에는 벽을 부숴버리는 것이기 때문에 3차원 부분의 값을 1로 바꿔준다.

## 애로 사항
1. 벽이 나올때마다 이전 3차원 좌표의 값을 1로 초기화해줬는데, 이게 오류의 원인이 되었다.
2. 일단 4방향 탐색을 끝낸 후에 3차원 좌표의 값을 1로 하여 탐색해야한다.
