# 9466 : 텀 프로젝트
## 코드 설명

1. cycle 형성 여부를 따져보는 문제(솔직히 이게 bfs인가..? 라는 생각이 듬)
2. state는 cycle 형성 여부를 체크하는 배열(현재 탐색하는 index로 사이클을 표시한다)
3. graph는 학생이 선택한 학생의 번호(입력 받은 값)을 담는 배열이다
4. state에 현재 탐색하는 index를 넣고, 배열 값에 따라 이동한다
5. 만약 state[cur] == index 과 같이 cycle을 발견한다면 state 배열의 값을 -1로 바꿔준다(cycle 형성 표시)
6. state[cur] != 0 과 같이 다른 cycle에 형성이 되었다면 바로 return을 해준다.

## 애로 사항
1. 이틀 동안 생각해봤는데 도저히 모르겠어서 답을 참고한 문제.. 답지를 봐도 그렇게 와닿지 않았다.
2. 진짜 이게 왜 bfs문제인지 모르겠다.
3. cycle발견시 바로 return을 해줘야 하는데 그렇지 못해 infinate loop 발생
