# 2573 : 빙산
## 코드 설명

1. melting(주변 빙산 있는거보고 없어지기) 후 check_segment(빙산 나눠졌는지 확인,bfs) 두개의 주 로직으로 구성
2. 처음에 check_segment로 빙산 몇개인지 확인 후 1개라면 melting으로 녹이기, 2개 이상이라면 출력


## 애로 사항
1. bw.write덕분에 3번은 더틀린것 같다.
2. 저번에 풀어봤을때는 deepCopy를 이용해서 항상 시간 초과가 났었다. 이번에 풀때는 tmp변수를 사용하여 새로운 graph에 대입하는 형태로 만들었더니
시간복잡도가 훨씬 줄어들었다.
3. check_segment에서 visited를 매번 초기화해주지 않아서 틀렸었다. 돌릴때마다 초기화시켜줘야한다.
